Setup Tasks

These tasks need to be completed to finish setting up and using your controller.

Task 1 - Initial Configuration
Task 2 - Device Adoption

Task 1 - Initial Configuration

Step 1. First connection

When you first connect to your controller follow the on-screen prompts to set it up.

We advise signing in with a UniFi cloud account from UniFi but you can create local users if you'd prefer.

Step 2. Update the inforum-url setting

This ensures your devices know how to talk to your controller.

  1. Inside your controller go to Settings -> System -> Advanced.
  2. Find "Inform Host" and set it the DNS name for the UniFiCentral controller. ie "unc.unificentral.app" not a full url.
  3. Your devices will now be able to see and talk to your controller.
Step 3. Good to go!

You are now ready to move onto device adoption (detailed below).

Task 2 - Device Adoption

You can follow the guide listed below, but please note only the methods listed below will work on a cloud controller.

  • UniFi Mobile App (we recommend this one!)
  • DHCP Options 43
  • DNS
  • Direct SSH and set inform-url
